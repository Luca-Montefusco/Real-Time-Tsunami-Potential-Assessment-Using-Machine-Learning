# Real-Time-Tsunami-Potential-Assessment-Using-Machine-Learning
Project completed during the third year of the B.Sc. in "Mathematical Sciences for Artificial Intelligence" for the Earthquake Physics for Machine Learning course.

Earthquakes and tsunamis represent some of the most destructive natural hazards affecting coastal regions worldwide. 
The Traditional Early Warning System (EWS) often struggle to predicts these events and to warn the population: this project aims to serve as powerful decision-support tools in the race againts time.
Using early waveforms and pre-event activity, we want to categorize tsunami risk: the best dataset to train a Machine Learning model with this purpose is the Japan Meteorological Agency (JMA) database, because is characterized by a vast, efficient seismic network. These datasets provide essential parameters, including tsunami detection and verification.
For this project, the approach used are:
- First part: Implementation of a Multi-Input LSTM network for simulated data
- Second part: Implementation of a class-weighted Random Forests for historical data, to assist the first mode
